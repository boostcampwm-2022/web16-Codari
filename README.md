## 프로젝트 소개 

공동 편집 기능을 만들고 싶다는 의지 하나로 진행된 프로젝트!!

실시간으로 다른 사람과 함께 문서를 편집해보세요.

🤷‍♂️ : Are you guys on the same page?

👨‍👨‍👦 : Yes, cause we are on Codocs!

<br />

## 실행 방법 

#### frontend 
```
npm run start:dev 
```

#### backend/socket 
```
npm run start:dev
```

#### backend/api
```
npm run start:dev
```
